Adding the ChatKit widget to your website is simple. Just include the widget script and initialize it:

Step 1: Include the Widget Script

<!-- Add this before closing </body> tag --> <script src="https://yoursite.com/js/chatkit-widget.js"></script>

Step 2: Initialize the Widget

<script> // Initialize the widget with your configuration LightwaveChat.init({ apiUrl: 'https://yoursite.com/api/chatkit', organizationId: 'your-organization-id', position: 'bottom-right', // bottom-left, top-right, top-left theme: 'lightwave', // Your custom theme autoOpen: false, // Auto-open after page load showNotification: true // Show notification badge }); </script>

Step 3: Customize (Optional)

// Programmatically control the widget LightwaveChat.open(); // Open the widget LightwaveChat.close(); // Close the widget LightwaveChat.toggle(); // Toggle open/close LightwaveChat.showNotification(); // Show notification badge // Send a message programmatically LightwaveChat.sendMessage('Hello, I need help'); // Check widget status console.log(LightwaveChat.isOpen()); // true/false console.log(LightwaveChat.isOnline()); // true/false
